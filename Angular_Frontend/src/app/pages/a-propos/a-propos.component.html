📘 À propos du projet

Ce projet a été développé dans le cadre du Master 2 Informatique de l’Institut Polytechnique de Saint-Louis, à l’Université Gaston Berger.

🎯 Objectif du projet

Détecter automatiquement les fraudes bancaires en analysant les caractéristiques des transactions à l’aide d’un modèle d’intelligence artificielle supervisé. L'interface web permet de visualiser les résultats des prédictions en temps réel.

🛠️ Technologies utilisées
  • Python (Machine Learning)
  • Flask (API REST)
  • TypeScript
  • Bootstrap 5 (interface responsive)
  • HTML5 / CSS3
  • Angular (frontend framework)
📦 Fonctionnalités principales
  • Formulaire dynamique pour envoyer les données des transactions
  • Affichage de prédiction avec indicateur de risque
  • Historique des prédictions avec filtres et recherche
  • Visualisation graphique du taux de fraude (Chart.js)
📚 Données utilisées

Le modèle a été entraîné sur un jeu de données contenant des informations de transactions bancaires simulées. Les variables ont été normalisées et le déséquilibre de classe a été géré avec des techniques telles que SMOTE.
